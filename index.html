Concepto de GIT

Git es un sistema de control de versiones. Lleva un registro de todos los cambios y avances de tu proyecto. Además, ayuda almacenar proyectos en la nube. De igual manera, ayuda a que varias personas trabajen en un mismo proyecto y pueden realizar modificaciones sin afectar a los demás archivos. Una vez que estén listos los cambios se fusionan con la rama principal.

PRINCIPALES COMANDOS DE GIT

  • GIT INIT:Este comando funciona para iniciar un proyecto con git.
  • GIT CONFIG --GLOBAL USER.NAME:Para ver el nombre del usuario
  • GIT CONFIG --GLOBAL USER.EMAIL:Para ver el correo del usuario
  • GIT ADD:Permite agregar cambios al área de reparación.
  • GIT STATUS:Muestra el estado del directorio de trabajo y del área del entorno de ensayo.
  • GIT COMMIT -M:Registra instantaneas de archivo de forma permanente en el historial de versión.
  • GIT LOG:Se utiliza para listar el historial de versiones de la rama actual.
  • GIT BRANCH:Para mirar la rama.
  • GIT CHECKOUT MAIN:Es la acción de moverse entre diferentes ramas.
  • GIT CHECKOUT -B MAINPermite cambiar el nombre de la rama
